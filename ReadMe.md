## 프로젝트 소개
- 동물병원, 미용실 예약을 한 곳에서 할 수 있는 플랫폼
- 마케팅, 매출 정산 등을 펫브리띵이 대신 해줌으로서 제휴한 업체들은 고객에게 집중해 좀 더 나은 서비스를 제공
- 고객과 업체가 직접 통화할 필요 없이 간편하게 문의할 수 있는 채팅 서비스 제공

## 프로젝트 기능구현
- 곽서현
  - 로그인, 예약, 결제, 찜

  - 포인트, 게시판, 문의
